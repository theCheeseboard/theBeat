---

# Dependencies
- Qt 5
  - Qt Core
  - Qt GUI
  - Qt Widgets
  - Qt Multimedia
  - Qt SVG
- [the-libs](https://github.com/vicr123/the-libs)

## Build
Run the following commands in your terminal. 
```
mkdir build
qmake ../theBeat.pro
make
```

## Install
On Linux, run the following command in your terminal (with superuser permissions)
```
make install
```
